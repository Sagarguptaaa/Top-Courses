## Run Locally

Clone the project

```bash
  git clone https://github.com/thesagarguptaa/Top-Courses.git
```

Go to the project directory

```bash
  cd Top-Courses
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
